## Installation

```shell
pipenv install  # install dependencies
pipenv shell  # activate virtual env
deactivate  # deactivate virtual env
exit  # exit project
```

## Implementation

- Remember to restart the server after making code changes!
- To run the code,

```shell
cd [directory_name]
python3 blockchain.py
```
